SOFTWARE REQUIREMENTS SPECIFICATION (SRS)

Project Name: Premium Men’s Salon App
Platform:  Android (Expo / React Native)
Backend Hosting: Railway
Database: PostgreSQL

1. INTRODUCTION
1.1 Purpose

This document defines the full requirements for a luxury men’s salon mobile application that provides appointment booking, live queue tracking, service catalog, reviews, work showcase, and customer engagement tools.

The system must be scalable to support:

Single salon

Multiple branches (future SaaS expansion)

High daily booking volume

1.2 Scope

The application will:

Allow customers to book appointments

Display real-time queue

Show service price list

Display work gallery (haircuts, beard styles)

Collect and display reviews

Provide business contact & about information

Provide admin dashboard to manage everything

Support elegant luxury branding

Be optimized for mobile UX

Be production scalable

3. FUNCTIONAL REQUIREMENTS
3.1 AUTHENTICATION
FR-1 Registration

Username: 10-digit phone number (unique)

Password (min 8 chars)

First name

Last name

Profile image (optional)

FR-2 Login

JWT authentication

Role-based routing

FR-3 Profile

Users can:

Update name

Update password

Upload profile image
Phone number cannot be changed.

3.2 APPOINTMENT MANAGEMENT
FR-4 Booking

Client can:

Select date

View available time slots

Select service

Confirm booking

System must:

Prevent double booking

Lock transaction during booking

Assign queue position

FR-5 Appointment Cancellation

Client can cancel future appointment

Admin can cancel any appointment

Optional cancellation time limit

FR-6 Live Queue

Client can:

See currently serving

See own queue position

See estimated wait time

Admin can:

Mark in service

Mark completed

Mark no-show

Reorder queue (drag & drop)

3.3 PRICE LIST MODULE
FR-7 Services Management

Admin can:

Add service

Edit service

Delete service

Each service contains:

Name

Description

Duration

Price

Category (Haircut / Beard / Combo / Premium)

Client can:

View services

Filter by category

3.4 REVIEW SYSTEM
FR-8 Reviews

Clients can:

Leave rating (1–5 stars)
