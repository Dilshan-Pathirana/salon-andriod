# SOFTWARE REQUIREMENTS SPECIFICATION (SRS)

## 1. Introduction

### 1.1 Purpose

This document defines the functional and non-functional requirements for a **Luxury Hair Salon Progressive Web Application (PWA)** built with:

* **Frontend**: React.js
* **Backend**: Node.js + Express.js
* **Database**: Firebase (Firestore)
* **Hosting**: Firebase Hosting
* **Authentication**: Firebase Authentication
* **Deployment**: PWA (Installable on Android & iOS)

The system supports customer appointment booking, queue handling, and administrative salon operations.

---

## 2. System Overview

### 2.1 Product Perspective

The application is:

* A Progressive Web App (PWA)
* Installable on Android & iOS
* Hosted on Firebase
* Uses RESTful APIs
* Role-based (Admin / User)

### 2.2 System Architecture

**Architecture Pattern:**
Client–Server + Firebase-managed services.

```
React PWA (Frontend)
        ↓
Node + Express API (Firebase Cloud Functions or Express Server)
        ↓
Firebase Firestore Database
        ↓
Firebase Authentication
        ↓
Firebase Hosting
```

---

* Firebase Storage

---

## 4.2.4 Session Management (Calendar System)

### Flow:

1. Admin selects date
2. Marks day as Open/Closed (default = open)
3. Sets open time & close time
4. Clicks "Initiate"
5. System generates time slots

Rules:

* Time slot duration = configurable (default 30 mins)
* Prevent overlapping bookings
* Closed days not bookable

---

## 4.2.5 Appointment Management

Admin can:

* View all appointments
* Add manual appointment
* Reschedule
* Cancel
* Change appointment order
* Move appointment up/down

Appointment Status:

* Pending
* Confirmed
* Completed
* Cancelled
* No-show

---

## 4.2.6 Queue Management

Admin can:

* View today's live queue
* Change order (drag/drop)

* Remove appointment

# COMPLETE API TABLE

---

## Authentication APIs

| Method | Endpoint | Description | Role |
| ------ | ------------------------- | --------------- | ------ |
| POST | /api/auth/register | Register user | Public |
| POST | /api/auth/login | Login | Public |
| POST | /api/auth/logout | Logout | User |
| POST | /api/auth/change-password | Change password | User |

---

## User APIs

| Method | Endpoint | Description | Role |
| ------ | ------------------ | -------------- | ----- |
| GET | /api/users | Get all users | Admin |
| GET | /api/users/:id | Get user | Admin |
| PUT | /api/users/:id | Update user | Admin |
| DELETE | /api/users/:id | Delete user | Admin |
| PUT | /api/users/profile | Update profile | User |

---

## Service APIs

| Method | Endpoint | Description | Role |
| ------ | ----------------- | ---------------- | ------ |
| POST | /api/services | Add service | Admin |
| GET | /api/services | Get all services | Public |
| PUT | /api/services/:id | Update service | Admin |
| DELETE | /api/services/:id | Delete service | Admin |

---

## Story APIs
